# macOS Tips

## Window Tiling

**Built-in keyboard shortcuts** (macOS 15 Sequoia+):
- `fn+ctrl+left` - Tile window to left half
- `fn+ctrl+right` - Tile window to right half
- `fn+ctrl+up` - Tile window to top half
- `fn+ctrl+down` - Tile window to bottom half
- `fn+ctrl+F` - Maximize window (full screen tiling)

Alternative: Hold the green maximize button to see tiling options.
